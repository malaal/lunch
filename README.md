## Lunch Today
### What's for lunch today?

Web-based lunch voting system, using SQLAlchemy/SQLite and CherryPy. Voting based on python-vote.

Totally a **work in progress** that currently doesn't do a lot.

**Requirements**:
* python-vote-full
* SQLAlchemy
* CherryPy
